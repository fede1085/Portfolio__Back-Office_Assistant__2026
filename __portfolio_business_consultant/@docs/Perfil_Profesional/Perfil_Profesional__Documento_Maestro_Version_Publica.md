# ✅ Usage Rules (versión adaptada)
1. Priorizar **Core Keywords** en el titular (Headline).  
2. En el About, integrar una mezcla natural de:  
   - identidad (AI Productivity Consultant)  
   - productividad  
   - sistemas organizativos  
   - IA práctica  
   - digital workspace  
3. Rotar herramientas sin sonar técnico: Notion, Workspace, Gemini, ChatGPT.  
4. Cada post de LinkedIn debe incluir **2–3 keywords** según el tema.  
5. Reforzar visibilidad en región incluyendo keywords de **Bélgica** y **Uruguay**.  
6. Actualizar este set cada 3–6 meses según evolución del mercado y de mi oferta.
